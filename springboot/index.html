Spring Boot has many many Plugins...

  • embedded servlet container (jetty, tomcat7, undertow)
  • logging
  • security
  • relational databases (jpa)
  • nosql databases (redis, mongodb)
  • search engines (solr, elastic search)
  • support for evolutionary database
  • production ready features (metrics, tracing, health status, database uplink status)
